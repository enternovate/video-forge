# Security Policy

## Privacy Commitment
Video Forge processes all media **locally on your device**. No data is ever transmitted to any server.

- **No telemetry** — no usage data, crash reports, or analytics collected
- **No accounts** — no sign-up, no login, no user tracking
- **No AI cloud services** — captions use the browser's Web Speech API locally
- **Royalty-free codecs** — only VP9/AV1/WebM are bundled. No H.264/H.265/MPEG-LA patented formats
- **Open source** — fully auditable under MIT license

## Codec Legal Status
| Codec | Status | Notes |
|-------|--------|-------|
| VP9 | Royalty-free | Bundled via WebCodecs API |
| AV1 | Royalty-free | Supported when browser supports |
| WebM | Royalty-free | Container format |
| H.264/AVC | Patent-encumbered | NOT bundled |
| H.265/HEVC | Patent-encumbered | NOT bundled |
| AAC | Patent-encumbered | NOT bundled |
